# Contributing To DieselAtlas

Contributions are welcome when they improve the truck maintenance reference without weakening safety or evidence boundaries.

## Data Requirements

- Use original, public-domain, permitted, or clearly licensed diagrams and text.
- Do not copy proprietary service-manual pages or manufacturer schematics.
- Identify the final official service-data authority and what each source proves.
- Include safety gates, phased workflow, limitations, measurement notes, and return-to-service checks.
- Use HTTPS evidence links and explicit units.
- Never infer installed hardware or service values from an identifier alone.

Submit changes to this repository or the canonical [Atlas suite](https://github.com/BuiltByDasilva/wrenchatlas). Safety-critical changes require source review and browser/package validation before release.
